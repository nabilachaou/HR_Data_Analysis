# HR Data Analysis

## Description  
Ce projet traite un jeu de données RH (Human Resources) afin de :  
- nettoyer et formater les données avec SQL ;  
- analyser démographiquement et professionnellement les employés (âge, genre, race, département, etc.) ;  
- visualiser les résultats dans un tableau de bord Power BI.

## Structure du projet  
- `SQL/` : scripts SQL de nettoyage et d’analyse (fichier `hr_data_cleaning.sql`).  
- `PowerBI/` : fichier `HR_Analysis.pbix` contenant le rapport Power BI.  
- `README.md` : ce document explicatif.  

## Technologies utilisées  
- MySQL ) pour les requêtes SQL.  
- Power BI Desktop pour la création des visualisations.  
- Games de données RH (âge, date d’embauche, terme, etc.).  
- SQL : utilisation de fonctions `STR_TO_DATE`, `DATE_FORMAT`, `TIMESTAMPDIFF`, etc.
